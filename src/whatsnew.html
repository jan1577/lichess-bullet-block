Custom Blitz Rules

You no longer have to choose between blocking all Blitz games or none. With the new Custom Rule option you can set exact thresholds:

  • Minutes ≥ — only allow Blitz games at or above this time control (3–15 min).
  • Increment ≥ — only allow games with at least this many seconds of increment (0–20 sec).
  • AND / OR — combine both conditions. AND means both must be met; OR means either one is enough.
Example Set Minutes ≥ 8 OR Increment ≥ 3 to allow any game with increment of 3+ seconds. This would allow 5+3 and 10+0, while blocking 3+0, 3+2, 5+0.

Preset Buttons

The extension now removes the small preset time format buttons below the sliders when starting a new lobby game.
